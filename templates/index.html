About this project

This is the Natural Language Processing model I built as part of my capstone project for the BrainStation Data Science bootcamp.

In short, what the model does is analyse the content of a news article about a football/soccer manager from the English Premier League and output a prediction if this article is about someone who is likely to be fired in the next days or not.

The model was built using mostly the Scikit-Learn and NLTK libraries for Python. The code used in this project can be found here.
